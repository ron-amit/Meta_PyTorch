# Meta_PyTorch



Requirements:
* Python 2.7+ (recomended package manager: Anaconda python)
* PyTorch 0.2+

Run files:
* main_standard.py         - Learn standard neural network.
* main_Bayes.py            - Learn stochastic neural network.
* Toy_Examples\Toy_Main.py -  Unsupervised 2D  regression.
